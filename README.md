# A Fast MSM Implemenation

## Usage:

* Install [Rust](https://www.rust-lang.org/tools/install):
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    source $HOME/.cargo/env
    ```

* Run Tests:
    ```bash
    cargo test
    ```

* Benchmarking:
    ```bash
    cargo bench
    ```
